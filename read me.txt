== Description ==
Description: Convert English Month and Number to Bengali Language

Change List:

Time, Date, AM,PM,Calender,Comment Number, Comment Date


== Installation ==

Just Upload `Sonar Bangla` to the `/wp-content/plugins/` directory
Activate the plugin through the 'Plugins' menu in WordPress
And enjoy this plugin.
